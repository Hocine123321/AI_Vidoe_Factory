# AI Video Factory

A PowerShell-based AI video generator application with multiple sub-scripts and modules.

## Project Structure

```
AI_Vidoe_Factory/
├── config/              # Configuration files
├── context/             # Context and data files
├── modules/             # PowerShell modules
├── output/              # Generated video output
├── python/              # Python scripts and dependencies
├── main.ps1            # Main entry point
├── settings_menu.ps1   # Settings and menu interface
├── error.log           # Error log
└── learning.log        # Learning/progress log
```

## Getting Started

1. Clone the repository
2. Run the main script:
   ```powershell
   .\main.ps1 -Action menu
   ```

## Features

- Video generation settings
- AI model configuration
- Output preferences
- Python environment integration
- Logging system

## Requirements

- PowerShell 5.0 or higher
- Python 3.x (for AI processing)
