# Reference Layout: 30-Second Cadence

## Format Rules
- Every block = one scene = one image = ~3 seconds.
- Timestamp format: [MM:SS - MM:SS]
- Label each block: Hook / Point 1 / Point 2 / Outro

## Example Script

[00:00 - 00:03] (Hook): ...Your operating system isn't slow. It's bloated by background tracking services.
[00:03 - 00:06] (Point 1): Every boot triggers hidden telemetry tasks wasting your CPU cycles.
[00:06 - 00:12] (Point 2): Those tasks silently transmit data while starving your foreground apps.
[00:12 - 00:18] (Point 3): Stripping them out immediately frees memory registers for real work.
[00:18 - 00:24] (Point 4): Users report 40% faster boot times after a single cleanup script.
[00:24 - 00:30] (Outro): To unlock your hardware's true ceiling, you need to strip the fat. now did you know that....[video loops]
