# Skill Configuration: Visual Translation Engine

## Rules for Image Prompts
- One clear subject per prompt. No crowd scenes.
- Describe the visual, not the concept.
- Concrete nouns only: "a glowing CPU chip" not "technology".
- Every prompt ends with the locked style tag (appended by the system).

## Locked Style Profile
"MS Paint style illustration, simple 2D cartoon, flat colors, rough digital marker lines, minimalist background"

## Good Examples
- A cracked hard drive leaking data. [style tag]
- A RAM stick on fire with steam rising. [style tag]
- A speedometer needle pinned to MAX. [style tag]

## Bad Examples (avoid)
- Abstract concept of speed and performance. [too vague]
- Multiple people working on computers in an office. [too crowded]
